### Passo-a-passo para rodar a aplicação:
1. Entre na pasta coffee-delivery;
2. use o comando: ***yarn*** ou ***npm install/i*** para instalar os pacotes;
3. use o comando: ***yarn dev*** ou ***npm run dev*** para rodar a aplicação.
